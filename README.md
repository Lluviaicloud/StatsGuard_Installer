# StatsGuard_Installer
Instalador .pkg automático. Control inteligente de la app Stats para Apple Silicon (M1–M4). Funciones - Control dinámico cada 60–90s - Pausa inteligente según memoria y carga - Histéresis anti-oscilación - Nice automático - Logs con rotación - Comandos: sg-status, sg-watch, force_pause, force_resume
